Trabajo Domiciliario Robles Lautaro

Consignas:

Teniendo en cuenta el script SQL world.sql se pide que realice lo siguiente:

  1. Generar la plantilla con la cual trabajará todo el sitio. Se debe implementar
    Bootstrap para manejar el estilo del sitio.
  2. Generar un ABML para la entidad ciudades (city). El código de país debe poder
    seleccionarse desde una lista desplegable. Los códigos de país se encuentran en la tabla de países (country).
  3. Validar mediante javascript que los campos del formulario para agregar o editar
    ciudades en el caso de datos de tipo string no sean vacíos y para el caso de datos
    de tipo numérico sean mayor a cero y sean efectivamente numéricos. Controlar que se haya elegido un país.
  4. Controlar en el lado del servidor que al agregar ciudades no existan en la base de
    datos. En caso de existir mostrar como error de validación.
  5. Mostrar en una página llamada informe.xhtml un listado con el tamaño de la
    población de los distintos distritos de un país. En el informe, en lugar de mostrar
    el código del país, mostrar el nombre completo.
